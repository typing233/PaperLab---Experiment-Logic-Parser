# PaperLab - 实验逻辑解析器

一款专为研究生设计的 AI 辅助阅读工具，能将晦涩复杂的学术实验文本一键转化为清晰的结构化逻辑模块与交互式思维导图。

## ✨ 核心功能

### 📋 9 大逻辑模块
自动将学术实验文本拆解为以下 9 个结构化模块：

1. **研究目的** - 一句话概括实验要做什么
2. **实验假设** - 核心假设是什么
3. **实验设计** - 设计类型 + 通俗描述
4. **材料与方法** - 实验对象、材料、流程步骤、关键技术
5. **关键变量** - 自变量、因变量、控制变量三栏展示
6. **实验结果** - 主要发现 + 数据要点
7. **核心结论** - 实验得出的核心结论
8. **局限性** - 实验的不足之处
9. **阅读建议** - 给研究生的实用阅读策略

### 🧠 交互式思维导图
- **可视化展示**：将 9 大模块以思维导图形式直观呈现
- **平移操作**：鼠标拖拽即可移动整个导图
- **缩放功能**：滚轮缩放或点击缩放按钮
- **全屏模式**：支持全屏查看，提升阅读体验
- **重置视图**：一键恢复默认视角

### 📖 故事化解读
- 用通俗易懂的故事化语言重新解读整个实验
- 让非专业人士也能轻松理解复杂的学术研究

### 📚 专业术语释义
- 自动提取文本中的关键专业术语
- 提供通俗解释，降低阅读门槛

### 💾 本地历史记录
- 自动保存解析历史（最多 20 条）
- 点击即可加载之前的解析结果
- 支持单条删除和全部清空

### 📋 一键复制功能
- 复制单模块内容（如故事化解读）
- 一键复制全部 9 大模块内容
- 复制内容自动格式化为 Markdown 格式

### 🔑 隐私安全
- API Key 仅保存在浏览器本地（localStorage）
- 明文文本和 API Key 不会上传到任何服务器
- 所有解析请求直接从浏览器发送到 DeepSeek API

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm 或 yarn

### 安装步骤

1. **安装依赖**
```bash
npm install
```

2. **启动服务**
```bash
npm start
```

3. **访问应用**
打开浏览器访问：`http://localhost:2359`

### 配置 API Key

1. 首次使用需要配置 DeepSeek API Key
2. 在页面顶部的「API 配置」区域输入您的 API Key
3. 点击「保存配置」按钮
4. API Key 将自动保存在浏览器本地

> 💡 **获取 API Key**：访问 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册并获取 API Key

## 📖 使用指南

### 输入文本
1. 在「输入实验文本」区域粘贴您的学术实验文本
2. 支持 100-10000 字的文本输入
3. 实时显示当前字数统计

### 开始解析
1. 确认 API Key 已配置
2. 点击「开始解析」按钮
3. 等待 AI 解析完成（通常需要 10-30 秒）

## 🔧 技术架构

### 后端
- **框架**：Express.js
- **HTTP 客户端**：Axios
- **端口**：2359

### 前端
- **原生技术**：HTML5 + CSS3 + JavaScript (ES6+)
- **存储**：LocalStorage（API Key 和历史记录）
- **绘图**：原生 SVG（无第三方库依赖）

### AI 服务
- **模型**：DeepSeek Chat (deepseek-chat)
- **API 端点**：https://api.deepseek.com/v1/chat/completions
- **响应格式**：JSON

## ❓ 常见问题

### Q1: 解析失败怎么办？
- 检查 API Key 是否正确
- 检查网络连接是否正常
- 确认 DeepSeek 账户有足够的额度
- 尝试减少输入文本长度

### Q2: 为什么提示文本长度不符合要求？
- 最少需要 100 字才能进行有效解析
- 最多支持 10000 字，超出部分可能被截断
- 如果文本过长，建议分段解析

### Q3: 思维导图无法正常显示？
- 请使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 确保 SVG 支持已启用
- 尝试刷新页面

### Q4: 历史记录丢失了？
- 历史记录存储在浏览器本地存储中
- 清除浏览器缓存或隐私数据会导致历史记录丢失
- 不同浏览器之间历史记录不共享

### Q5: API Key 安全吗？
- API Key 仅存储在您的浏览器本地
- 不会被发送到任何第三方服务器
- 只有在调用 DeepSeek API 时才会使用
- 您可以随时清除已保存的 API Key
